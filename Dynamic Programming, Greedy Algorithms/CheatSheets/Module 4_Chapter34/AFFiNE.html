Conjunctive Normal Form (CNF)
Updated 27 Mar 2024
Info
Heading 2
What is CNF?
Type '/' for commands
Think of CNF as a specific way to organize or write down a list of your choices or decisions, but with some rules on how you can format that list. Specifically, CNF is about writing these choices in a way that emphasizes **"AND"**s and **"OR"**s.
Heading 3
Breaking Down the Components
Literal: This is the most basic element and represents a simple, straightforward choice or statement. It could be something like "I will go to the gym today" (let's call this a) or "I will not eat junk food today" (we can represent the negation as b, where b is eating junk food).
Disjunction (): This is a fancy term for "OR". It's used when you're okay with one thing or another happening. For example, "I will go to the gym today OR I will go for a run" can be written as a c (assuming c is going for a run).
Conjunction (): This is just an "AND". It's used when you're saying both things need to happen. If you're planning to "go to the gym AND eat healthy today", it's like saying a b.
Heading 3
Rules of CNF
Conjunction of Disjunctions: Your entire decision list (formula) needs to be a big "AND" of smaller "OR" groups.
No Deep Nesting: You can't have complex layers upon layers. It's like saying, "I can't have, 'If it's sunny, either I'll go to the gym unless it's too crowded, in which case I'll run, unless...' and so on." CNF doesn't allow for that level of complexity.
Straightforward Negations: You can say "I will not eat junk food" but not "It's not true that I won't not go to the gym" because that's confusing (double negation).
Heading 3
Examples and Explanations
Simple choice: a (Going to the gym)
Not doing something: b (Not eating junk food)
Doing two things: a b (Going to the gym AND eating healthy)
A choice between two plans, but you must pick at least one plan: (a b) (c d) translates to "Either go to the gym or not eat junk food, AND either go for a run or read a book."
Heading 3
Why CNF Matters
Type '/' for commands
Transforming any decision-making process into CNF can make it easier to analyze or process with logic tools (like computers). However, trying to reformat complex decisions into CNF can sometimes make the decision list way longer or more complicated than it started.
Heading 2
What is 3CNF SAT?
Type '/' for commands
First,3CNFis a specific type of CNF where each disjunction (the "OR" part) is limited to exactly three literals. Think of it as a strict diet plan where each meal (disjunction) can only consist of up to three food items (literals).
Type '/' for commands
SATstands forsatisfiability. When we talk about 3CNF SAT, we're dealing with the question: "Is there a way to make the entire CNF formula true by assigning true/false values to the literals in a way that satisfies all the conditions?
Heading 2
The Challenge of 3CNF SAT
Type '/' for commands
3CNF SAT is known to be NP-complete. This means that, as of now, no one knows a way to quickly solve every instance of this problem as the number of meals (clauses) grows. It's like trying to plan an entire year's worth of meals under strict rules and wanting to know if the plan is even possible. For small numbers of meals, it's doable; for a whole year, it might take a lifetime to figure out.
